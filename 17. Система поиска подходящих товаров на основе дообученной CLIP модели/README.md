# Поиск товаров по тексту на дообученном CLIP


Дообучение мультимодальной модели CLIP на каталоге одежды и построение системы поиска изображений товаров по текстовому запросу (text → image retrieval).

## Цели

1. Дообучить CLIP на датасете товаров магазинов одежды.
2. Реализовать поиск изображений по текстовому запросу.

Целевая метрика — средний **CLIP score > 30** (косинусная близость вектора картинки и её описания × 100).

## Данные

- **44 441** запись: изображение товара + текстовое описание (`image`, `description`).
- Очистка: удалены пустые описания (281 шт.), дубликаты и строки без файла картинки.
- Разбивка: **90% train / 10% test**.
- Описания часто длиннее лимита CLIP (77 токенов) - в рамках проекта хвост обрезался (основная суть в начале).

## Подход

- Модель: предобученный `CLIPModel` (~**151M** параметров), изображения масштабируются до 224×224.
- Свой класс `FashionDataset` (пара картинка + текст); токенизация и препроцессинг батчем в `collate_fn` через `CLIPProcessor`.
- Обучение: стандартный **контрастивный лосс** CLIP (симметричная кросс-энтропия по матрице сходств, `return_loss=True`), оптимизатор AdamW, cosine-шедулер с warmup, 4 эпохи.
- Валидация: средний CLIP score на тесте в конце каждой эпохи; сохранение лучшего чекпоинта (`clip_best`).
- Поисковая система: один раз считаются эмбеддинги всех картинок, запрос кодируется в вектор, ранжирование по косинусной близости, выдача топ-K.

## Итоги

- До дообучения Средний CLIP score **29.86** 
- После дообучения Средний CLIP score **32.66**

- Целевой порог **30 взят** (val 32.66); обучение шло стабильно, лосс вышел на плато за 3–4 эпохи.
- Модель «из коробки» уже понимала товары (стартовый score ~29.86), дообучение уверенно перевело её через порог.
- Поиск по запросам (`red skirt`, `blue sunglasses` и т. п.) возвращает релевантные товары.
- Главный резерв качества — не лишние эпохи, а **работа с текстами описаний**: модель видела только начало описаний из-за обрезки до 77 токенов.
- Идея на развитие: предварительно суммаризировать длинные описания отдельной моделью, чтобы укладывать суть в лимит CLIP.
